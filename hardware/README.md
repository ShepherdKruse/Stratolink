# Hardware Design

PCB layouts, circuit diagrams, and mechanical designs for Stratolink pico-balloon hardware.

## Directory Structure

- `/pcb` - PCB design files (KiCad, Altium, Eagle, etc.)
- `/circuits` - Circuit schematics and diagrams
- `/3d-models` - 3D enclosure and mechanical designs
- `/docs` - Hardware documentation and specifications

## Hardware Components

1. RAK3172 LoRaWAN Module (STM32WLE5)
2. u-blox GNSS Module
3. Power Management Circuit
4. Battery System
5. Environmental Sensors (optional)
