# Stratolink Brand Assets

## Logo

Place the Stratolink logo file here as `logo.png` or `logo.svg`.

The logo should be:
- Circular design with "STRATOLINK" text
- "GLOBAL PICO-CONNECTIVITY" tagline
- Hot air balloon with signal waves
- Globe representation

## Usage

The logo is used in:
- README.md (project documentation)
- Web application (dashboard and branding)
- Documentation and presentations

## File Formats

Preferred formats:
- `logo.png` - For general use (transparent background recommended)
- `logo.svg` - For scalable vector graphics
- `logo-dark.png` - Dark mode variant (if applicable)
