# Web Application Images

Place web-optimized versions of the Stratolink logo here for use in the Next.js application.

## Recommended Files

- `logo.png` - Main logo (transparent background, ~400x400px)
- `logo-icon.png` - Favicon/icon version (32x32px or 64x64px)
- `logo-dark.png` - Dark mode variant (if applicable)

These images are served from `/images/` in the web application.
